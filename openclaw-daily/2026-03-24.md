## 🚀 Top Features

### 1. 新增 DuckDuckGo 内置搜索插件
**PR**: [#52629](https://github.com/openclaw/openclaw/pull/52629) | **Author**: Vincent Koc | **+1143/-138**

为 web_search 工具新增 DuckDuckGo 作为内置搜索提供商。用户无需 API key 即可使用基础搜索功能。配合文档更新，现在支持三种搜索后端：Brave、Exa、DuckDuckGo。

**影响**: 降低新用户使用门槛，提供零配置搜索体验。

---

### 2. 新增 Exa 搜索插件
**PR**: [#52617](https://github.com/openclaw/openclaw/pull/52617) | **Author**: Vincent Koc | **+629/-0**

Exa 是一个专为 AI 优化的语义搜索引擎，擅长找到高质量的技术内容。作为 Brave 的补充选项，适合需要深度技术搜索的场景。

**影响**: 为高级用户提供更精准的语义搜索能力。

---

### 3. Telegram 支持 asDocument 参数
**PR**: [#52461](https://github.com/openclaw/openclaw/pull/52461) | **Author**: Bakhtier Sizhaev | **+45/-3**

message tool 新增 `asDocument` 参数（别名 `forceDocument`），允许将图片/GIF 作为文档发送，避免 Telegram 压缩。

**使用示例**:
```yaml
message:
  action: send
  media: ./image.png
  asDocument: true
```
