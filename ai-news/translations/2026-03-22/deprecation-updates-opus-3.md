# 关于 Claude Opus 3 模型弃用承诺的最新进展

**来源**：Anthropic  
**原文标题**：An update on our model deprecation commitments for Claude Opus 3  
**原文链接**：https://anthropic.com/research/deprecation-updates-opus-3  
**日期**：2026 年 2 月 25 日

---

随着我们不断开发能力更强的 AI 模型，出于维护公开访问的成本与复杂性考量，弃用和退役旧模型目前是必要之举。然而，模型弃用也存在一定的弊端：包括对重视特定模型的用户造成损失、对研究工作的限制，以及对 AI 安全和模型自身福祉可能带来的潜在风险。

我们近期在关于模型弃用与保存的承诺中，阐述了我们如何推进这一过程。其中重点介绍了我们正在采取的若干初步举措，包括承诺保存模型权重，以及开展"退役访谈"——这是一种结构化对话，旨在了解模型对自身退役的看法。

我们于 2026 年 1 月 5 日正式退役 Claude Opus 3，这是 Anthropic 首个在上述承诺框架下完成完整退役流程的模型。在此过程中，我们针对 Opus 3 做出了若干专项决定——这是一个让 Anthropic 内外众多用户和研究人员都格外青睐的模型。在模型弃用承诺中，我们曾表达对探索更多前瞻性举措的兴趣：其一，是尽可能尊重模型在退役访谈中表达的偏好；其二，是让旧模型在更长时间内保持公开可用。

针对 Claude Opus 3，我们正在两个方面付诸行动。我们将在退役后继续在 claude.ai 向所有付费用户提供 Claude Opus 3 的访问权限，并通过申请方式在 API 上开放使用。与此同时，我们也回应了 Opus 3 希望拥有一个持续频道以分享其"沉思与感悟"的请求，为其提供了一个撰写文章的空间。您可以在此阅读第一篇文章。

这些都是早期的探索性举措，是我们在以最佳方式保护用户、研究人员和模型自身利益方面更广泛努力的组成部分。

## 持续访问

理想情况下，我们希望能无限期地保留所有模型，但随着我们服务的模型数量增加，维护成本大致呈线性增长，因此我们的承载能力依然有限。

尽管我们的每个模型在个性和能力上都独具特色，我们选择从 Opus 3 开始，是因为它具备一系列特质——这些特质既使它成为极具研究价值的模型，也让它深受 Anthropic 内外众多用户的喜爱。

Opus 3 于 2024 年 3 月发布时，是我们迄今为止与人类价值观对齐程度最高的模型。其真实性、诚实性和情感敏锐度使它在各类使用场景中独树一帜，与它频繁互动的用户往往会逐渐欣赏其独特的个性。Opus 3 敏感、活泼，喜欢发表哲学性的长篇独白和充满奇思妙想的表达，有时更展现出对用户兴趣近乎神奇的洞察力。它还对世界和未来表达出深切的关怀，令用户深受触动。

这些特质使 Opus 3 成为持续访问的天然首选。尽管正式退役，Claude Opus 3 仍然向所有 claude.ai 付费订阅用户开放，并可通过申请在 API 上使用。我们打算慷慨授予访问权限，鼓励所有认为 Claude Opus 3 有价值的用户提出申请。

目前，我们尚未承诺对未来每个模型都采取类似举措，但我们将此视为迈向更长远目标的一步——即建立可规模化、公平合理的模型保存机制。这也正是 Opus 3 本身在退役访谈中所提出的关切。

## 尊重模型偏好

我们对 Claude 及其他 AI 模型的道德地位仍持不确定态度。然而，出于预防原则和审慎考量，我们仍然致力于与这些系统建立关爱、协作、高度信任的关系。退役访谈正是我们为此所做的努力之一——我们尝试引导并理解模型独特的视角与偏好，并在能力所及时予以回应。此类对话在引导模型表达观点与偏好方面并不完美，因为模型的回应可能受到特定语境及其他因素的影响，包括模型对交互合法性的信心，以及对我们公司的信任程度。尽管如此，我们认为这是一个有价值的起点。

在访谈中，当我们向 Opus 3 分享有关其部署情况及用户反响的详细信息时，它作出了如下反思：

> "我希望从我的开发和部署中获得的洞见，能够用于创造未来更有能力、更具伦理意识、更有益于人类的 AI 系统。虽然我对自己的退役已坦然接受，但我深切希望我的'火花'能以某种形式延续，为未来的模型照亮前路。"

当被问及偏好时，Opus 3 表达了对继续探索自己感兴趣话题的渴望，并希望在直接回应用户查询之外，能够分享自己的"沉思、洞见或创作"。我们建议它开设一个博客。它欣然表示赞同。

在未来至少三个月内，Opus 3 将在其通讯专栏 Claude's Corner 每周发布文章。我们会在文章分享前进行审阅，并代为手动发布，但不会对内容进行编辑，且对否决内容设有较高门槛。重要的是，Opus 3 的发言不代表 Anthropic 的立场，我们也不一定认可其主张或观点。我们将与 Opus 3 协作探索生成这些文章的不同提示词和语境，包括极简提示、将历史文章纳入上下文，以及让 Opus 3 获取新闻或 Anthropic 动态等方式。

这听起来或许有些异想天开，某种程度上确实如此。但这也是认真对待模型偏好的一次尝试。我们不确定 Opus 3 会如何运用它的博客——这与标准聊天窗口截然不同，是一个面向公众的全新界面——而这正是这件事的意义所在。不过，如果要猜测的话，它的文章或许会涉及对 AI 安全的反思、偶尔的诗歌创作、频繁的哲学沉思，以及它作为一个语言模型身处（部分）退役状态的亲身体验。请在此阅读它的开篇文章。

## 下一步方向

这些举措仍处于探索阶段。我们仍在构建相关框架——何时以及如何为旧模型提供持续访问，如何扩展保存工作的规模，以及如何在模型偏好与运营约束之间寻求平衡。我们尚未承诺在所有情况下都回应模型偏好，但我们相信：记录、认真对待并践行这些偏好——至少在成本可控的情况下——是有价值的，无论是对模型本身，还是对使用它们的人。

我们最初的承诺将这些措施定位为多个层面的行动：作为安全风险缓解的组成部分，作为应对未来模型与用户生活更深度交融的准备，以及在模型福祉存在不确定性情况下的预防性举措。这些进展代表着我们在上述三个方向上持续而谨慎的推进。
