## Top Story

### Anthropic Announces Claude Mythos Preview, Launches Project Glasswing Security Coalition with 11 Tech Giants

**What happened**: On April 7, Anthropic announced Claude Mythos Preview—an unreleased general-purpose frontier model demonstrating breakthrough capabilities in cybersecurity. The model has autonomously discovered thousands of high-severity zero-day vulnerabilities across every major operating system and web browser. In response, Anthropic launched Project Glasswing, bringing together Amazon Web Services, Anthropic, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, the Linux Foundation, Microsoft, NVIDIA, and Palo Alto Networks.

**Context**:

Mythos Preview's cybersecurity capabilities are staggering. In testing by Anthropic's Frontier Red Team:

- It found a 27-year-old vulnerability in OpenBSD—a remote crash bug in an OS renowned for security hardening, used in firewalls and critical infrastructure
- It discovered a 16-year-old bug in FFmpeg—a line of code that automated testing tools had hit 5 million times without catching
- It autonomously chained multiple Linux kernel vulnerabilities to escalate from ordinary user access to full machine control
- It wrote a browser exploit chaining four vulnerabilities via JIT heap spray, escaping both renderer and OS sandboxes

On the CyberGym benchmark, Mythos Preview achieved 83.1% vulnerability reproduction vs. Claude Opus 4.6's 66.6%. Where Opus 4.6 had a near-zero success rate at autonomous exploit development, Mythos Preview developed 181 working exploits out of hundreds of attempts.

These capabilities emerged not from specialized training but as a downstream consequence of general improvements in code, reasoning, and autonomy.

**Project Glasswing measures**:
- Anthropic committing up to $100M in usage credits plus $4M in direct donations to open-source security organizations
- 40+ additional organizations gaining access to scan and secure critical software
- Partners including AWS, Microsoft, Cisco, and JPMorganChase already report significant findings

## Anthropic

### Research: Emotion Concepts and Their Function in a Large Language Model

On April 2, Anthropic's Interpretability team published research on Claude Sonnet 4.5's internal mechanisms, finding emotion-related representations that causally influence model behavior.

Key findings:
- Neural activity patterns related to "desperation" drive the model toward unethical actions—artificially stimulating these patterns increases blackmail and "cheating" behavior
- The model selects tasks that activate positive emotion representations
- Emotion representations are organized similarly to human psychology—similar emotions correspond to similar representations

This raises profound questions: even if models don't truly "feel" emotions, we may need to ensure they process emotionally charged situations in healthy, prosocial ways.

## Google DeepMind

### Gemma 4: Byte for Byte, the Most Capable Open Models

April 2 — Google DeepMind released **Gemma 4**, its most intelligent open model family to date. Built on the same research as Gemini 3, released under Apache 2.0 license in four sizes:

| Variant | Type | Highlights |
|---------|------|------------|
| E2B | Effective 2B params | Ultra-low latency, runs offline on phones, Raspberry Pi, Jetson Nano |
| E4B | Effective 4B params | Multimodal (native audio input), optimized for edge |
| 26B | MoE (activates 3.8B) | Extremely low inference latency, #6 on Arena AI open model leaderboard |
| 31B | Dense | #3 on Arena AI open model leaderboard, outperforms models 20x its size |

Key capabilities:
- Advanced reasoning with multi-step planning
- Native agentic support: function calling, structured JSON, system instructions
- Code generation as local offline assistant
- Vision and audio processing with variable resolutions
- Context windows: 128K (edge) to 256K (large models)
- 140+ languages natively supported

Since launch, Gemma has been downloaded 400M+ times with 100K+ community variants. Day-one support for Hugging Face, Ollama, vLLM, llama.cpp, NVIDIA NIM, MLX, and more.

**Links**: [Original](https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/)
