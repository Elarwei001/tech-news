# Introducing Claude Design by Anthropic Labs

- 原文链接：https://www.anthropic.com/news/claude-design-anthropic-labs
- 发布日期：2026-04-17
- 译注：以下为基于原文的完整中文翻译，保留产品名、组织名与专有名词。

今天，我们推出 Claude Design。这是 [Anthropic Labs](https://www.anthropic.com/news/introducing-anthropic-labs) 的一款新产品，让你可以与 Claude 协作，创建精致的视觉作品，例如设计稿、原型、幻灯片、单页文档等。

Claude Design 由我们能力最强的视觉模型 [Claude Opus 4.7](https://www.anthropic.com/news/claude-opus-4-7) 提供支持，目前以 research preview 形式向 Claude Pro、Max、Team 和 Enterprise 订阅用户开放。我们会在今天逐步向用户推出。

即使是经验丰富的设计师，也必须克制探索范围，因为他们很少有时间去原型化十几种不同方向，因此通常只能把自己限制在少数几种方案内。而对于创始人、产品经理和市场人员来说，即便他们有想法，如果没有设计背景，要把这些想法做出来并分享出去也可能是件很困难的事。

Claude Design 为设计师提供了更大范围探索的空间，也让其他人能够产出视觉作品。你只需要描述自己的需求，Claude 就会先做出第一个版本。然后，你可以通过对话、内联评论、直接编辑，或者使用由 Claude 创建的自定义滑块，不断调整直到满意为止。在获得相应访问权限后，Claude 还可以把你团队的设计系统自动应用到每一个项目上，让输出结果与公司其他设计保持一致。

团队已经在使用 Claude Design 来完成以下工作：

- 真实感原型：设计师可以把静态设计稿变成易于分享的交互式原型，用于收集反馈和做用户测试，而不需要经过代码审查或提交 PR。
- 产品线框图和设计稿：产品经理可以快速画出功能流程，并交给 Claude Code 实现，或者分享给设计师做进一步打磨。
- 设计探索：设计师可以迅速生成多种不同方向，用于探索。
- 融资 deck 和演示文稿：创始人和客户经理可以在几分钟内从粗略提纲生成一套完整、符合品牌风格的演示文稿，然后导出为 PPTX 或发送到 Canva。
- 营销素材：市场人员可以制作落地页、社交媒体素材和营销视觉内容，然后再让设计师进一步润色。
- 前沿设计：任何人都可以构建带有语音、视频、着色器、3D 和内置 AI 的代码驱动型原型。

## 它如何工作

Claude Design 遵循一种自然的创作流程。

将你的品牌融入其中。在 onboarding 过程中，Claude 会通过读取你的代码库和设计文件，为你的团队构建一个设计系统。此后每个项目都会自动使用你的配色、字体和组件。你可以随着时间推移不断完善这个系统，团队也可以维护多个设计系统。

从任何地方导入。你可以从一个文本提示开始，也可以上传图片和文档（DOCX、PPTX、XLSX），或者让 Claude 查看你的代码库。你也可以使用网页捕获工具，直接从网站抓取元素，让原型看起来更接近真实产品。

用细粒度控制进行迭代。你可以对特定元素做内联评论、直接编辑文本，或者使用调节旋钮实时微调间距、颜色和布局。然后再让 Claude 把这些改动应用到整个设计中。

协作。设计文档支持组织范围内共享。你可以将某个文档设为私有，也可以通过链接分享给组织中任何人查看，或者授予编辑权限，让同事能够修改设计，并在群组对话中一起与 Claude 协作。

导出到任何地方。你可以将设计作为组织内部 URL 分享、保存为文件夹，或者导出到 Canva、PDF、PPTX 或独立 HTML 文件。

交接给 Claude Code。当设计准备好进入实现阶段时，Claude 会把所有内容打包成一个 handoff bundle，你只需一条指令就可以把它交给 Claude Code。

在接下来的几周里，我们会让构建 Claude Design 集成变得更容易，这样你就可以把它连接到更多团队已经在使用的工具中。

过去几年里，我们非常喜欢与 Anthropic 合作，我们双方都深深关注如何把复杂的事情变简单。在 Canva，我们的使命一直是赋能全世界去设计，这意味着要把 Canva 带到创意开始发生的任何地方。我们很高兴在与 Claude 的合作基础上继续前进，让人们能够顺畅地把 Claude Design 中的想法和草稿带入 Canva，在那里它们会立即变成完全可编辑、可协作的设计，随时可以继续打磨、分享和发布。

Brilliant 复杂的交互和动画历来都很难做原型，但 Claude Design 把静态设计转成交互式原型的能力，对我们来说是一次质变。我们最复杂的页面，在其他工具里往往需要 20 多次提示才能重建，而在 Claude Design 里只需要 2 次提示。把设计意图也包含在 Claude Code 的交接里，让从原型到生产环境的跃迁变得非常顺畅。

Claude Design 让我们团队的原型制作速度大幅提升，甚至可以在对话过程中直接现场做设计。我们现在能够在所有人离开会议室之前，就把一个粗糙想法变成可运行的原型，而且输出结果仍然忠于我们的品牌和设计规范。过去需要一周时间在 brief、mockup 和评审轮次之间来回沟通的流程，现在在一次对话里就能完成。

## 开始使用

Claude Design 面向 Claude Pro、Max、Team 和 Enterprise 订阅用户开放。访问权限已包含在订阅计划中，并会消耗你的订阅额度。若启用 [extra usage](https://support.claude.com/en/articles/12429409-manage-extra-usage-for-paid-claude-plans)，还可以在超出额度后继续使用。

对于 Enterprise 组织，Claude Design 默认关闭。管理员可以在 [Organization settings](https://support.claude.com/en/articles/14604406-claude-design-admin-guide-for-team-and-enterprise-plans) 中启用它。

现在就前往 [claude.ai/design](http://claude.ai/redirect/website.v1.f9ae731b-8b80-4321-bc1c-93c4064973cd/design) 开始设计。
