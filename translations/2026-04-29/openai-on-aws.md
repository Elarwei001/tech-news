# OpenAI models, Codex, and Managed Agents come to AWS

> AI 译，完整翻译自 OpenAI 官方原文：<https://openai.com/index/openai-on-aws/>

今天，OpenAI 与 AWS 正在扩大双方的战略合作关系，帮助企业在其 AWS 环境中使用 OpenAI 的能力进行构建。我们很高兴让 AWS 客户能够访问最好的前沿模型、agent 和工具，并且这些能力可以运行在他们已经在使用的系统、安全协议、合规要求和工作流之中。

此次与 Amazon 扩展后的合作集中在三个关键领域，今天都以 limited preview 形式推出：

- AWS 上的 OpenAI 模型
- AWS 上的 Codex
- 由 OpenAI 提供支持的 Amazon Bedrock Managed Agents

这些能力合在一起，为组织提供了更多在应用开发、软件工程和 agentic 工作流中使用 OpenAI 的方式，同时又能建立在他们已经在 AWS 中采用的基础设施、安全、治理和采购流程之上。

对于许多公司来说，要大规模使用 AI，就意味着要把最好的模型带到团队已经在使用的系统里。这也是为什么我们要在 Amazon Bedrock 上推出 OpenAI 模型，包括我们最强的前沿模型 GPT-5.5。

客户现在可以在 AWS 中使用 OpenAI 模型进行构建，同时继续沿用他们已经依赖的服务、安全控制、身份系统和采购流程。

对于开发者来说，这意味着他们在使用 OpenAI 进行构建时拥有更大的灵活性，无论是开发新的 AI 应用、把智能嵌入现有产品，还是构建能够推理、采取行动并支持更复杂业务流程的 agentic 工作流。

对于企业来说，这意味着从试验到生产之间有了一条清晰、统一的路径，因为 OpenAI 的能力可以直接出现在他们最重要工作负载已经运行的 AWS 环境中。

现在，每周已有超过 400 万人使用 Codex，团队正在把它应用到软件开发生命周期的各个环节，包括编写代码、解释系统、重构应用、生成测试、改造遗留代码库，以及加速一系列超出编码本身的专业工作流。越来越多的团队也在用 Codex 加速研究、分析和基于文档的工作，通过连接他们每天都在使用的应用和工具，从总结源材料到生成简报、幻灯片和电子表格。

Codex 是 OpenAI 的前沿 coding harness 与产品套件，而现在组织可以使用直接由 Amazon Bedrock 提供的 OpenAI 模型来驱动 Codex。这意味着任何拥有 AWS 承诺和 Bedrock 访问权限的公司，都可以几乎无摩擦地开始使用 OpenAI 强大的 coding agent 与相关产品。

客户只需要将 Codex 配置为使用 Bedrock 作为 provider 即可开始。这为客户带来了他们所期待的 AWS 企业级特性，包括安全、计费和高可用性。所有客户数据都由 Amazon Bedrock 处理，符合条件的客户还可以将 Codex 的使用计入他们的 AWS cloud commitments。

Codex on Bedrock 目前以 limited preview 形式提供。客户可以通过 Bedrock API 将 Codex 配置为使用 Amazon Bedrock，首批支持包括 Codex CLI、Codex 桌面应用以及 Visual Studio Code 扩展。

我们同时还推出了由 OpenAI 提供支持的 [Amazon Bedrock Managed Agents](https://aws.amazon.com/bedrock/managed-agents-openai/)，为企业提供一种在其可信 AWS 环境中部署高级 agents 的新方式。

借助 Bedrock Managed Agents，组织可以构建能够保持上下文、执行多步骤工作流、使用工具，并在复杂业务流程中采取行动的 agents。这有助于客户更快从实验走向生产，同时让 agent 开发始终与他们所期待的基础设施、安全和运营标准保持一致。

对于企业来说，Bedrock Managed Agents 让团队可以把注意力放在如何让 agents 真正对工作有用，而不是去拼装部署、工具使用、编排与治理所需的基础设施。它把更困难的部分处理掉了，并且内建与 Amazon 安全和合规控制的集成。结果就是，企业可以更快地把能在真实企业环境中运行的 agents，从原型推进到生产。

我们与 Amazon 的战略合作，目标是帮助组织以生产级规模部署先进 AI。随着 OpenAI 模型、Codex 和 Managed Agents 进入 Amazon Bedrock，客户拥有了一条更快、更安全的路径，能够把 AI 真正应用到他们的业务中。我们很期待看到大家的组织会构建出什么。
