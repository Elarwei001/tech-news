# Introducing Claude Design by Anthropic Labs

> 中文翻译，原文来自 Anthropic 官方博客  
> 原文链接: https://www.anthropic.com/news/claude-design-anthropic-labs

今天，我们推出 Claude Design。这是一款全新的 [Anthropic Labs](https://www.anthropic.com/news/introducing-anthropic-labs) 产品，让你可以与 Claude 协作，创建精致的视觉作品，例如设计稿、原型、幻灯片、单页材料等。

Claude Design 由我们能力最强的视觉模型 [Claude Opus 4.7](https://www.anthropic.com/news/claude-opus-4-7) 提供支持，目前以研究预览版形式向 Claude Pro、Max、Team 和 Enterprise 订阅用户开放。我们会在今天陆续向用户逐步推出。

即使是经验丰富的设计师，也必须克制探索的范围，因为他们几乎没有时间去原型化十几种方向，因此往往只能从少数几种方案中选择。而对于创始人、产品经理和市场人员来说，即使脑中有想法，如果没有设计背景，要把它创作出来并分享给别人，依然很困难。

Claude Design 既给设计师更大的探索空间，也让其他人有能力制作视觉作品。你只需要描述需求，Claude 就会先生成一个初版。之后，你可以通过对话、行内评论、直接编辑，或由 Claude 创建的自定义滑杆不断微调，直到满意为止。在获得访问权限后，Claude 还可以自动将你团队的设计系统应用到每个项目中，从而让输出结果与公司其余设计保持一致。

团队已经在用 Claude Design 完成以下工作：

- **真实原型**：设计师可以把静态 mockup 变成易于分享的交互式原型，用于收集反馈和用户测试，而无需代码审查或提交 PR。
- **产品线框图与设计稿**：产品经理可以先勾勒功能流程，再交给 Claude Code 落地实现，或者分享给设计师继续打磨。
- **设计探索**：设计师可以快速产出多种方向，进行大范围探索。
- **路演 deck 与演示文稿**：创始人和客户经理可以在几分钟内从粗略大纲生成完整、符合品牌风格的演示文稿，然后导出为 PPTX 或发送到 Canva。
- **营销素材**：市场团队可以创建落地页、社交媒体素材和营销视觉内容，再让设计师进一步润色。
- **前沿设计**：任何人都可以构建由代码驱动的原型，包含语音、视频、着色器、3D 和内置 AI。

## 工作方式

Claude Design 遵循一种自然的创意流程。

**你的品牌，直接内置。** 在初始设置期间，Claude 会通过读取你的代码库和设计文件，为团队构建设计系统。之后的每个项目都会自动使用你的配色、字体和组件。你可以随着时间推移持续完善这套系统，团队也可以维护多套设计系统。

**从任何地方导入。** 你可以从文本提示开始，也可以上传图片和文档（DOCX、PPTX、XLSX），或者直接让 Claude 访问你的代码库。你还可以使用网页抓取工具，直接从你的网站提取元素，让原型更接近真实产品。

**用细粒度控制进行微调。** 你可以对特定元素添加行内评论、直接编辑文本，或者使用调节旋钮实时修改间距、颜色和布局。然后再让 Claude 将这些更改应用到整个设计中。

**协作。** 设计文件支持组织范围内共享。你可以将文档保持私密，或者设置为组织内任何拥有链接的人都可查看，也可以授予编辑权限，让同事一起修改设计，并在群聊中共同与 Claude 协作。

**导出到任何地方。** 你可以将设计作为组织内 URL 共享、保存为文件夹，或者导出为 Canva、PDF、PPTX 或独立 HTML 文件。

**交接给 Claude Code。** 当设计准备进入开发阶段时，Claude 会将所有内容打包为一个交接包，你只需一条指令就能把它交给 Claude Code。

在未来几周内，我们会让 Claude Design 更容易构建集成，以便你可以把它连接到团队已经在使用的更多工具。

我们在过去几年一直非常享受与 Anthropic 的合作，并且彼此都深度关注如何把复杂的事情做得简单。在 Canva，我们的使命始终是赋能全世界去设计，而这意味着要把 Canva 带到创意开始萌芽的任何地方。我们很高兴能在与 Claude 的合作基础上继续推进，让人们能够无缝地把 Claude Design 中的想法和草稿带入 Canva，并立刻变成可完全编辑、可协作的设计，继续打磨、分享和发布。

Brilliant 的复杂交互与动画，以往一直非常难以做原型，但 Claude Design 能把静态设计转成交互式原型，这对我们来说是一次质变。我们那些最复杂的页面，在其他工具里需要 20 多个提示才能重现，而在 Claude Design 里只用了 2 个提示。并且，在把设计意图一并交给 Claude Code 后，从原型到生产的过渡变得非常顺畅。

Claude Design 让我们团队的原型制作速度大幅提升，甚至能在讨论过程中现场完成设计。我们可以在所有人离开会议室之前，就从一个粗略想法走到一个可运行原型，而且输出结果仍然忠实于我们的品牌和设计规范。过去需要一整周、在 brief、mockup 和多轮评审之间来回折腾的流程，现在在一次对话里就能完成。

## 开始使用

Claude Design 面向 Claude Pro、Max、Team 和 Enterprise 订阅用户开放。该功能已包含在你的订阅套餐中，并会计入套餐使用额度。如果想在额度之外继续使用，也可以启用 [extra usage](https://support.claude.com/en/articles/12429409-manage-extra-usage-for-paid-claude-plans)。

对于 Enterprise 组织，Claude Design 默认关闭。管理员可以在 [Organization settings](https://support.claude.com/en/articles/14604406-claude-design-admin-guide-for-team-and-enterprise-plans) 中启用它。

你可以前往 [claude.ai/design](http://claude.ai/redirect/website.v1.4098c51f-b051-4b00-83a6-d16f16f6be75/design) 开始设计。
