# GPT-5.5 Instant：更聪明、更清晰，也更个性化

原文链接：https://openai.com/index/gpt-5-5-instant/
发布日期：2026-05-05

> 说明：以下为对 OpenAI 官方文章的完整中文翻译，保留信息结构、数据表述与产品含义。

我们正在更新 ChatGPT 的默认模型，并向所有用户开放，让它变得更聪明、更准确，回答也更清晰、更简洁，并且更贴合你个人。

因为 Instant 是数亿人日常使用的默认模型，哪怕是小改进也会带来很大差别。这次更新让日常互动变得更有用、也更愉快：在各类主题上给出更强、更紧凑的回答，拥有更自然的对话语气，并且当个性化有帮助时，能更好地利用你已经分享过的上下文。

现在，Instant 变得更可靠，在各方面的事实准确性都有显著改进，在准确性最重要的领域提升尤为明显。在内部评测中，GPT-5.5 Instant 在医学、法律和金融等高风险提示上的幻觉性错误声明，比 GPT-5.3 Instant 减少了 52.5%。在用户曾标记为存在事实错误的特别困难对话中，它也将不准确声明减少了 37.3%。

GPT-5.5 Instant 作为一个整体也更聪明了，在各类日常任务上的能力更强，包括分析照片和图像上传、回答与 STEM 相关的问题，以及判断何时应使用 web search 来提供更有帮助的回答。

这些改进也体现在视觉推理、数学和科学等评测项目的提升上。

通过这次更新，模型的回答会更紧凑、更直截了当，同时不丢失内容厚度，并保留让 ChatGPT 使用起来更愉快的温暖感和个性。它可以传递与以往模型相同的信息，但通常更有用，同时减少那些会让回答显得过长的冗长表达和过度格式化。它也会减少不必要的追问，并避免一些会让回答显得杂乱的东西，比如多余的 emoji。

现在，如果你已连接相关数据源，Instant 也更擅长利用过去聊天、文件和 Gmail 中的上下文，让回答感觉更贴近你个人，同时仍然由你掌控。它会更智能地判断什么时候额外的个性化能提升回答质量，也能更快搜索过去的对话以找到合适上下文，因此你不必像以前那样频繁重复自己。这对获取定制化建议、计划，或继续推进长期工作尤其有帮助。

我们还在所有 ChatGPT 模型中引入 memory sources，让你能看到回答是使用了哪些上下文来进行个性化，并配备新的控制项。当某个回答是经过个性化处理时，你可以看到它使用了哪些上下文，比如保存的记忆或过去的聊天，也可以在某些内容已经过时或不再相关时将其删除或纠正。

如果你选择分享聊天，memory sources 不会展示给其他人。你仍然掌控哪些内容存在于你的记忆中：你可以删除那些不希望再被引用的聊天，在设置中删除或修改保存的记忆，或使用不会使用也不会更新记忆的临时聊天。

memory sources 的设计目标，是让个性化更容易理解，但它们可能不会展示影响答案形成的每一个因素。例如，它可能只会在来源中展示部分最相关的过去聊天，而不是它检索和参考过的全部聊天。我们会持续改进这个视图，让它随着时间推移变得更全面。

GPT-5.5 Instant 从今天开始向所有 ChatGPT 用户逐步推出，取代 GPT-5.3 Instant 成为默认模型，并在 API 中以 chat-latest 的形式提供。对于付费用户，GPT-5.3 Instant 还会保留三个月，可通过模型配置设置访问，之后将退役。

基于过去聊天、文件和已连接 Gmail 的增强个性化能力，正在向网页端 Plus 和 Pro 用户推出，并将很快登陆移动端，未来几周内还计划扩展到 Free、Go、Business 和 Enterprise。memory sources 正在向所有 ChatGPT 消费者套餐的网页端推出，并将很快登陆移动端。具体个性化来源的可用性可能因地区而异。
