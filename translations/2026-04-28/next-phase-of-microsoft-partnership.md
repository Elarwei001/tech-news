# The next phase of the Microsoft OpenAI partnership

- 原文链接：https://openai.com/index/next-phase-of-microsoft-partnership/
- 发布日期：2026-04-27
- 说明：以下为基于 OpenAI 官方原文的完整中文翻译。

---

修订后的协议为长期合作提供了更清晰的框架。

创新节奏正在快速加快，这要求我们持续演进双方合作关系，以更好地服务客户并让两家公司都能受益。今天，我们宣布一项修订后的协议，用以简化我们的合作关系和合作方式，其基础是灵活性、确定性，以及更广泛地交付 AI 价值的共同目标。修订后的协议带来更高的可预期性，也强化了我们共同大规模构建和运营 AI 平台的能力，同时让双方都保有追求新机会所需的灵活性。协议明确包括以下几点：

- 微软仍将是 OpenAI 的主要云合作伙伴（primary cloud partner），OpenAI 的产品仍将首先在 Azure 上发布，除非微软无法并且选择不支持所需能力。OpenAI 现在也可以通过任何云服务商向客户提供其全部产品。
- 微软将继续持有对 OpenAI 模型与产品相关知识产权的许可，期限持续至 2032 年。该许可现改为非排他性（non-exclusive）。
- 微软将不再向 OpenAI 支付收入分成（revenue share）。
- OpenAI 向微软支付的收入分成将继续持续到 2030 年，不取决于 OpenAI 的技术进展，分成比例维持不变，但将受总额上限约束。
- 微软将继续作为主要股东，直接参与 OpenAI 的成长。

虽然这项修订简化了双方合作结构，但我们共同推进的工作仍然雄心勃勃。从扩展以吉瓦为单位的新数据中心容量，到在下一代芯片上展开合作，再到应用 AI 推动网络安全进步等，我们都很期待继续携手，推动 AI 在全球范围内前进并实现规模化落地，服务个人与组织。

## 译后说明

这篇公告篇幅不长，但信号很强。它并不是在宣布双方“降温”，而是在重写一套更适合 agent 与企业平台时代的合作边界。对 OpenAI 而言，更重要的是争取跨云分发与产品自主权；对微软而言，更重要的是保留 Azure 主场、股权收益和长期平台协同。
